Instruções para o Turista de Cabalgata:
Antes de montar:
- "Bem-vindos! Antes de começar, vamos repassar algumas instruções de segurança."
- "Por favor, usem o capacete durante todo o passeio. É obrigatório."
- "Este cavalo se chama...... Ele é muito dócil."
- "Para montar, coloque o pé esquerdo no estribo e impulsione-se para cima."
- "Segurem as rédeas com as duas mãos, mas sem puxar com muita força."
Durante a cavalgada:
- "Mantenham uma distância segura do cavalo da frente. Cerca de um cavalo de distância."
- "Se quiserem que o cavalo pare, puxem as rédeas suavemente para trás.
- "Para virar à direita, puxem a rédea direita. Para virar à esquerda, a rédea esquerda."
- "Se o cavalo começar a trotar e você se sentir inseguro, puxe as rédeas para trás suavemente."
- "Não gritem ou façam movimentos bruscos, isso pode assustar os cavalos."
- "Aproveitem a paisagem! O Vale do Maipo é lindo."
O que o Turista Pode Dizer ou Perguntar:
- "Como faço para o cavalo andar?" (Para fazer o cavalo andar, basta dar um leve toque com os calcanhares na barriga do cavalo).
- "Meu cavalo não quer andar. ¿O que eu faço?"
- "Como faço para ele parar?"
- "Socorro, ele está indo muito rápido!"
- "Posso tirar fotos enquanto estou no cavalo?"
- "Tem problema se eu soltar uma das mãos das rédeas?"
- "Onde fica o banheiro mais próximo?"
